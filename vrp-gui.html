Quest Archive GUI (RcloneBrowser)

This app lets you download directly from the server. Currently, only Sponsored Mirrors can use Quest Archive GUI.

Setup

  1. Exclude the folder from antivirus scanning.
  2. Extract Quest Archive GUI to the folder.
  3. Place your vrp.download.config in the folder.
  4. Run RcloneBrowser.exe.

How to Use

Double-click a mirror to find games. Right-click and press Download to choose a destination.
