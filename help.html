How to Help Out

Thank you for realizing it takes a lot of work to run this service. We are all volunteers that pay out of pocket for servers and games to crack.

Supporting the Servers

It takes funds to keep servers up and running. If you'd like to donate via crypto, our Wallets and Accepted currencies are listed on the Quest Archive Wallets page.

Donating Games

Contributing games helps a lot! Donating is as simple as loading Rookie, which will automatically ask you if you'd like to upload the game to us. You can also use the "Share Selected App" button to save the game manually.
